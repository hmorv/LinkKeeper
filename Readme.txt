INTRODUCTION
------------
SJO-LinkKeeper is a web application where you can store and classify your bookmarks. 
The user is able to create his own categories, and then store and manage the desired links on each category.
The application accepts new users by register, and it also shows the public Ip from the user.


 * For a full view, visit the project page:
   https://hugomoragues.com/LinkKeeper/index.php


REQUIREMENTS
------------
This application doesn't require nothing for the final user, except an account.
For the install on a server you will need:

 * A FTP to send the files to server
 * A server
 * If u want to manage the database, access to PHPMyAdmin of this page is needed.


INSTALLATION
------------
 * Copy the files to your public directory on the server. Use a FTP.

Warning: Don't move any files between directories, they are placed on the right ones.


 * Access to PHPMyAdmin and Import or copy the content of the database. (db.sql)

 * Enjoy!
